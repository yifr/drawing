Instructions for creating new images and descriptions


In some phases, you will need to create new images and descriptions.

You will have already seen images from the experiment. For these tasks, you will see a blank sketchpad and blank textbox like below. Your task is to draw a new image like the kinds of images you have seen, and write a short and clear description for your image. For example, you may draw a new image and a new description like:

Your new image here:

Please write a description for the image.
